Detailed Description

The SpericalCoord class defines a position, in spherical coordinates. \notes spherical coordinates are using right-handed notation
rho is distance.
theta is angle from z axis, clockwise.
phi is angle from x axis, counterclockwise
We use the notation that is common practice in physics, As specified by ISO standard 31-11.

Conversions from spherical to cartesian are performed using formulas indicated in http://mathworld.wolfram.com/SphericalCoordinates.html But, since we are using physics convention instead of math convention (as Wolfram), swap theta and phi from document.

With x looking forward: Theta: -: Up +: Down Phi: + Left -: Right
