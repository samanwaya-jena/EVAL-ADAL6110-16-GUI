ReceiverSimulatorCapture Class Reference

TheReceiverSimulatorCapture class is a specialized implementation of the ReceiverCapture that provides simulated data without any extanal device attached. \Remarks Often useful for demos or debugging purposes. Since it does not manage connections to the device, it is a poor example for development, though. More...

Detailed Description

TheReceiverSimulatorCapture class is a specialized implementation of the ReceiverCapture that provides simulated data without any extanal device attached. \Remarks Often useful for demos or debugging purposes. Since it does not manage connections to the device, it is a poor example for development, though.

Author
Jean-Yves Deschnes
