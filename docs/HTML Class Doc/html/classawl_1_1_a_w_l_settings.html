Detailed Description

Manages storage, reading and writing of all settings contained in the configuration file \Note In an application, there is typically only one instance of the AWLSettings Settings class. So typical usage is Constructor, InitSettings(), ReadSettings(). Then access to the variable through GetGlobalSettings().
