Detailed Description

CameraCalibration parameters are used to convert from CameraCoordinates to pixel coordinates in the image.

\reference: http://docs.opencv.org/doc/tutorials/calib3d/camera_calibration/camera_calibration.html

\Notes We use a calibration system that uses parameters similar to the OpenCV distorsion matrixes and camera calibration matrix. This means we can use the results from OpenCV calibration procedures in configuration files.
